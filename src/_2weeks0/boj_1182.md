### 접근 방식
- 완전 탐색의 기본 유형 중 하나인 부분 집합을 이용하는 문제다.
- i번째 index의 수를 포함하거나 제외하며 재귀 함수를 호출해주면 된다.
    ```java
    void recursive(int n, int s, int[] nums, int idx, int sum) {
        if (idx == n) {
            if (s == sum) {
                answer++;
            }
            return;
        }

        recursive(n, s, nums, idx + 1, sum + nums[idx]);
        recursive(n, s, nums, idx + 1, sum);
    }
    ```

### 시간 복잡도
- O(2^N)

### 공간 복잡도
- O(N)
