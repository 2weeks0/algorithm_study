### 접근 방식

- 소수 판별
  ```java
  boolean isPrime(int n) {
        if (n <= 1) {
            return false;
        }
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (n % i == 0) {
                return false;
            } 
        }
        return true;
   }  
  ```
  
- 팰린드롬 판단
  ```java
  boolean isPalindrome(int n) {
      String str = String.valueOf(n);
      for (int i = 0; i < str.length() / 2; i++) {
          if (str.charAt(i) != str.charAt(str.length() - 1 - i)) {
              return false;
          }
      }
      return true;
  }
  ```
  
### 시간 복잡도
- O(n^(1/2) * log10(N))

### 공간 복잡도
- O(1)
